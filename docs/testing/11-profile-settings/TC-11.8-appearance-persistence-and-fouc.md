# TASK-TC-11.8: Run TC-11.8 - Appearance persistence and FOUC

## Result: PASS

`index.html:56-69` runs the theme bootstrap synchronously before first paint:

```js
var t = localStorage.getItem('theme');
if (t === 'dark' || t === 'light') {
  document.documentElement.setAttribute('data-theme', t);
} else if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
  document.documentElement.setAttribute('data-theme', 'dark');
}
```

Choosing dark in `/settings/appearance` calls `theme.setMode('dark')` → writes
`localStorage.theme="dark"` and updates `<html data-theme="dark">`. On reload, the bootstrap
script restores the attribute before any rendering — no FOUC.
